Create A Map

The map creation will be achieved by the previously set _odom_ frame system and LiDAR scans. Two external packages are involved: rplidar_ros and slam_toolbox

1. Bring Up HomeR Driver Node

$ ros2 launch homer_control bringup_homer.launch.py

2. Joystick Guided Mapping

$ ros2 launch homer_control joy_chart.launch.py
